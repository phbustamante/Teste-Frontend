# SoulTech Front-End Challenge

Aplicação React com autenticação e gerenciamento de produtos.

## Instalação Rápida

```bash
git clone https://github.com/seuusuario/soultech-auth-app
cd soultech-auth-app
npm install
```

## Execução

```bash
npm start
```

Acesse: http://localhost:3000

## Requisitos
- Node.js (14.0.0+)
- npm
